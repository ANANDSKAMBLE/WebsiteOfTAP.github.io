HTML
HyeperText Markup Language is the standard markup
language for creating web pages and web applications.
In the module,you are going to learn about HTML tags, elements
and attributes,images,Rables,Links,Lists,Forms.

CSS
CSS basic styles
In this module, you would learn about Styling Images,Links
,Tables,Borders,Margins,Lists,Padding,Cursors.

JavaScript
In this module you will get full information about Styling Images,Links,
provides dynamic interactivity onwebsites.We'll show how to
add some basic Javascript funtionalities to your pages Like
printing a welcome messange using Javascript,single and multi
line comments using changing a CSS property using Javascript.

Bootstrap
Bootstrap themes are packages of HTML,CSS and Javascript
code that provide styling.UI components and pages layouts for
you to use in a web project.In essence,they are pre-built website
templates for you to adopt and build upon.In this module,you
are going to learn about the theming.

Java SE
This module helps you to understand the concepts of object- 
oriented programming.The module covers introductory elements
of OOP's including inheritance,
polymorphism,encapsulation,abstarction,aggegation add
advanced conceptslike error handling,file handling and 
multithreading.collection frameworks,maps.

JDBC
Implementing JDBC in the Spring Framework 
takes care of working with many low-level 
operations (opening/closing connections, 
executing SQL queries, etc.). JDBC in Spring has
several classes (several approaches) for interacting
with the database. The most common of these is using
the JDBC Template class. This is the base class that 
manages the processing of all events and database 
connections. In this module, we would learn 
how to implement JDBC with the JdbcTemplate class in
the Spring framework.

JEE
Overview of JEE Architecture, Introduction Remote Method Invocation, Sessions,
Creating an Application page, Cookies, Authentication with Session Control, Handle
errors and exceptions in Servlets Application, Implement inter-Servlets 
Communication, Develop JSP Applications, Web Application Basics.
